# 🌟 Vinayak Pal - Portfolio

A stunning dark-mode portfolio showcasing my expertise as an Azure DevOps/MLOps Engineer with 4.5 years of experience in cloud architecture, automation, and AI/ML deployments.

## ✨ Features

- 🎨 **Dark Mode Design** - Sleek, modern dark theme with vibrant purple/blue gradients
- 🎯 **Interactive 3D Elements** - Animated backgrounds, floating particles, and smooth transitions
- 🎮 **Draggable Skills** - Interactive skill cards you can move around the canvas
- 📱 **Fully Responsive** - Looks great on all devices
- ⚡ **Performance Optimized** - Built with React, Vite, and Tailwind CSS
- 🚀 **Motion Animations** - Smooth, professional animations throughout

## 🛠️ Tech Stack

- **Framework:** React 18 with TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS v4
- **Animations:** Motion (Framer Motion)
- **Drag & Drop:** react-dnd
- **Icons:** Lucide React
- **UI Components:** Radix UI

## 🚀 Quick Start

### Prerequisites
- Node.js 18+ installed
- pnpm package manager

### Installation

1. Clone the repository:
```bash
git clone https://github.com/vinayak_pal/portfolio.git
cd portfolio
```

2. Install dependencies:
```bash
pnpm install
```

3. Start development server:
```bash
pnpm run dev
```

4. Build for production:
```bash
pnpm run build
```
